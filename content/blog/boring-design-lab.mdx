從 2024 年 6 月畢業到入伍前，我在這裡待了約 8 個月的時間。雖說是實習生，但我的工作內容基本上與正職無異，參與的專案範疇涵蓋單頁式網站切版、多頁活動網站與官網開發，以及內部工具的研究與開發，範圍相當廣。不只用到了我熟悉的 React 和 Tailwind CSS，還碰到了 Vue、Angular、Firebase 等前端技術，拓展了自己在網頁前端與動態互動領域的視野。

## 隨時跟上時代的變化

BORING 團隊一直走在技術最前端，平常用 Notion、Slack、Figma 還有自架 NAS 伺服器來協作。我加入的 2024 年，剛好是 AI 產品大爆發的一年。大家在發想點子時會用 GPT 輔助，在寫簡報時讓 GPT 幫忙潤飾內容，甚至在開發時也會搭配 Cursor 或 GitHub Copilot 來提升 coding 效率。

剛開始我有點驚訝，畢竟學校老師還禁止我們用 ChatGPT 寫作業，許多傳統公司也會擔心資安問題而不允許使用 AI 工具。但 BORING 是個新創的科技團隊， Sam 的觀點非常務實，如果設計師或工程師們能在保持自己思維的同時，利用這些 AI 工具讓工作效率顯著的提升，又何樂而不為呢？

之前我在傳統產業的製造廠 IT 部門兼職，那裡的工作環境得用著老舊的電腦，網路也鎖了一堆網站，讓人做事時難免綁手綁腳。來到了 BORING，能夠自由探索最新技術，對我來說是很有幫助的。

## 不持續學習就會被淘汰

儘管我的工作能力在入職時是有符合 JD 的要求，但我來到 Boring 後還是學到了很多新技術。因為實習生不會因為沒經驗就被侷限在簡單的任務，每個專案都充滿新的挑戰。

有一次，某個專案因為需客戶的要求所以是採用 Angular 開發，而我在被指派要幫忙維護的任務前，其實根本沒接觸過 Angular。但其實由於前端框架在概念都有相似之處，而且也是使用 Tailwind CSS，所以我得以在短時間內學會 Angular 的語法並順利完成功能的維護。

不只如此，我還獲得了很多學習的機會：

- 在 IT 讀書會與同事們一起研究最新的技術筆記
- 透過 Code review 獲得同事的建議
- 觀看公司買的 Three.js journey 課程，並實際應用 3D 技術到專案當中
- 因為公司想要訓練自己的產圖模型，所以直接買了一套電腦當作 AI 工作站

這些經驗讓我認知到，身為 IT 團隊的一員最重要的不是會多少技術，而是保有「自學力」與「解決問題」的能力。因為這個時代真的變化的太快，會常常遇到自己沒碰過的主題，因此最重要的是能夠把問題解決，才是團隊中需要的人。資訊人才是無法一招吃遍天下的，也感謝主管 Pofu 常常引導我學習的方向，讓我少走很多冤枉路。

除了工作上的能力之外，BORING 的同事們也很多才多藝，大家會彈鋼琴、泡咖啡、講多國語言、畫畫、做遊戲……非常豐富，我現在才 22 歲，所以下一步呢？應該是會來學日文！

## 透明公開、樂於分享的文化

BORING 團隊的另一個特色就是「樂於分享」，由於目前公司是個十幾人的小團隊，扁平化的管理方式讓同事彼此間的溝通非常順暢。比如說工程師在與設計師協作時，若是發現設計稿上有難以實作或是沒考慮到的狀況，都可以馬上進行討論，以免浪費更多時間在解決衍生問題上面。

另外就是我們在每天開早會時，老闆 Sam 也時常會在會議中分享他的所見所想，不管是跟客戶談話中獲得到的心得，亦或是自己深思熟慮過後做的領導決策，他都會開誠布公地分享給大家。這讓整個團隊的凝聚力變得很強，每個人都能理解公司的願景，而不是只是埋頭做事，也能營造出鼓勵分享的公司文化。

還有每年一度的「 Error 大會」，在這段時間每個部門與員工都會反思自己與他人的優缺點，並給予鼓勵與建議。我也是透過這個機會，開始反省自己進入公司幾個月學到哪些東西以及還有哪裡不足。像是我當時自己紀錄的某個缺點，也有被其他同事提到，我也認真的紀錄下來並勉勵自己別再發生類似的錯誤，持續的改善自己。

## 結語

剛進公司時，我就得以負責開發 AFD 這樣的大型專案，之後接觸到的專案也都是像台新銀行、星宇航空這樣子的大客戶，甚至於很多專案是完全由我獨立開發完成的，我很訝異這是身為一名實習生可以接觸到的。另外在 2024 年， 公司也舉辦了 Work From New City，所有一起去日本旅行並回顧這年來的工作趣事，我發現到 BORING 的大家不只是專業人士，更是一群懂得享受生活、熱愛探索的人。 我想，正是這樣的個性，讓我們的作品都充滿創意、想法與生命力。

在 BORING 實習的這段時間，對我來說就像是一場驚奇連連的旅程。非常感謝有這段實習經驗，讓我不管是對於設計領域的美感都有所提升，或是在 Coding 方面有更實際的開發經驗，也讓我更清楚自己未來的方向，承蒙大家的照顧了！
