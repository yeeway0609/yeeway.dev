## 關於這個網站

此網站是用 Next.js 15 和 React 19 打造的，UI 樣式是基於 shadcn/ui 設計系統打造，CSS 框架理所當然是 Tailwind CSS，風格方面則主要參考了 [Cali](https://cali.so/) 和 [Bartosz Jarocki](https://jarocki.me/) 幾位大大的網站。

在內容管理上方面，我是直接用 Next.js 的 MDX 功能來撰寫，文章格式與資料夾結構可參考 [GitHub](https://github.com/yeeway0609/yeeway.dev/tree/main/src/content)，另外為了處理檔案編譯與 metadata 解析也在配置檔與社群套件花了不少功夫。會想用 MDX 主要是因為身為前端工程師，有時候會想加一些酷酷的小元件！像是 Code block、Rich link 或圖片牆......等等。

而目前寫部落格的心態還很佛系🤣 因為我自己還是很愛用社群媒體，所以部落格上主要會想紀錄一些重要事件的回顧，或是筆記分享，之後也會把平常在 Notion 整理的技術筆記或工具書籤庫整理上來！
