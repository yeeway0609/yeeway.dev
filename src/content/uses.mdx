## 🌐 瀏覽器 & 插件

- **Arc**
  - 最愛的功能為分頁即書籤、Space、Zap
  - 在手機上使用Arc search，可以釘選網站分頁之後在電腦上看
- **Safari**
  - 看 Netflix，因為 netflix 在 mac 的 chrome 上只有1080p
  - 看 YouTube，因為 Chrome 上的擋廣告插件幾乎都被yt 封鎖了，但在Safari 用 AdGuard 還是擋的掉
  - 手機上看英文文章搭配沈浸式翻譯，因為手機沒辦法使用 Chrome extension
- **Chrome** - 在用 Linux 時的上述替代品
- <RichLink href="https://immersivetranslate.com" imgSrc="https://s.immersivetranslate.com/static/official-static/img/twitter-image-zh-TW.jpg">沈浸式翻譯</RichLink> - 原文跟譯文會分段同步顯示，也可以只翻譯特定的段落

## 🧑‍💻 開發工具

- **VS Code + GitHub Copilot Pro** - 我自己比起 Cursor 更喜歡 Copilot Pro，因為我並不常使用 Agent 模式，Copilot 的 inline suggestion 就很有幫助了。而且 Copilot 支援的模型更多，也有包含網頁版的多種客戶端

- **Cloudfare** - 我的網域、DNS、CDN 圖床都在這裡
