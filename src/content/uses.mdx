## 🦾 軟體

- **Notion**  
  我是 Notion 魔人，database + page 合體技太好用了
- **Spotify**  
  不用多說了，串流王者
- **ChatGPT**  
  日常使用，用了 Gemini、Perplexity 之後，還是覺得 ChatGPT 最方便好用
- **Claude**  
  寫 code 時用，Artifact 拿來預覽網頁前端很方便，目前 Copilot 用的模型也都選 Claude Sonnet 4.5
- <RichLink href="https://github.com/RSSNext/Folo" imgSrc="https://app.follow.is/og-image.png">Folo</RichLink>  
  功能齊全的 RSS 閱讀器，不像 Feedly 一樣只能訂閱有 RSS feed 的電子報，搭配了 RSSHub 也能夠追蹤 Twitter 等社群平台的動態，我甚至還幫忙貢獻了[繁中介面翻譯](https://github.com/RSSNext/Folo/pull/3838)。
- <RichLink href="https://github.com/bitwarden" imgSrc="https://bitwarden.com/assets/7MxCY8VttLPQLbByiotlug/912d4b50c0ddccbb93478c0d1fad3fe8/bitwarden.com.png">Bitwarden</RichLink>
  開源的密碼管理器
- <RichLink href="https://bitly.com" imgSrc="https://app.follow.is/og-image.png">Bitly</RichLink>
  短網址服務，可以免費自訂網址後綴
- <RichLink href="https://www.raycast.com" imgSrc="https://www.raycast.com/opengraph-image-pwu6ef.png?7385e23163a01717">Raycast</RichLink>
  推爆！用鍵盤就能叫出一站式的捷徑工具，安裝之後我就刪除了好幾個 app 與 browser extension，最常用計算機、剪貼簿、色票吸取器、檔案壓縮、轉檔、Iconify、Spotify Player......等功能
- <RichLink href="https://imageoptim.com/mac" imgSrc="https://opengraph.githubassets.com/5f9baaf9514eaf5d54d65f2fc1e527cdd54349dcfe8369bf97e1868933f78d34/ImageOptim/ImageOptim">ImageOptim</RichLink>

- <RichLink href="https://shottr.cc/" imgSrc="https://shottr.cc/assets/page-preview.jpg">Shottr</RichLink>  
  螢幕截圖工具，內建 OCR、重複截圖、馬賽克......等功能
- <RichLink href="https://dropoverapp.com/" imgSrc="https://is1-ssl.mzstatic.com/image/thumb/Purple221/v4/44/44/78/44447802-f6b8-dca0-4ef8-aaec6de68a71/AppIcon-0-85-220-0-5-0-0-2x-0-0.png/1200x630bb.png">Dropover</RichLink>

- <RichLink href="https://moze.app/" imgSrc="https://images2.imgbox.com/c3/80/KuQEDTUI_o.jpg">Moze</RichLink>  
  UI 介面好看的記帳 App
- <RichLink href="https://ebook.hyread.com.tw/Template/store/hyreadApp3/index.jsp" imgSrc="https://ebook.hyread.com.tw/Template/store/hyreadApp3/images/BG-1.png">Hyread</RichLink>

## 🌐 瀏覽器 & 插件

- **Arc**  
  最愛的功能為分頁即書籤與 Spaces 的功能，在手機上也能透過 Arc search 釘選網站分頁之後在電腦上看
- **Brave**  
  看 YouTube 可以禁廣告，就我所知其他的 ad block 插件幾乎被 yt 封鎖了，剩他好用
- <RichLink href="https://immersivetranslate.com" imgSrc="https://s.immersivetranslate.com/static/official-static/img/twitter-image-zh-TW.jpg">沈浸式翻譯</RichLink>

## 🧑‍💻 開發工具

- **VS Code + GitHub Copilot Pro**  
  我自己比起 Cursor 更喜歡 Copilot Pro，因為我並不常使用 Agent 模式，Copilot 的 inline suggestion 就很有幫助了。而且 Copilot 支援的模型更多，也有包含網頁版的多種客戶端
- **Homebrew**  
  Mac 必備的套件管理工具
- **Cloudfare**  
  我的網域、DNS、CDN 圖床都在這裡
- **CodePen**  
  網頁前端工程師必備吧，附上我的[帳號連結](https://codepen.io/yeeway0609)
- <RichLink href="https://www.warp.dev" imgSrc="https://leading-bell-3e1c02e64d.media.strapiapp.com/hero_og_1017_0fe1efb432.png">Warp</RichLink>  
  體驗有點像 IDE 的終端機，也可以儲存常用的指令 snippet
- <RichLink href="https://www.beekeeperstudio.io/" imgSrc="https://opengraph.githubassets.com/dcc1e8677ba2855599ed497fceacdd770fb9be9267240dd99ed8b37e578bf066/beekeeper-studio/beekeeper-studio">Beekeeper Studio</RichLink>  
  開源的 SQL 資料庫管理工具，UI 比其他老牌工具好看非常非常多
- <RichLink href="https://httpie.io/" imgSrc="https://httpie.io/Images/Share/default.png">HTTPie</RichLink>  
  比 postman 更輕量的 API 測試工具，而且 UI 也比較簡潔好看，支援 CLI 與 GUI
- <RichLink href="https://iconify.design/" imgSrc="https://iconify.design/assets/og/iconify.png">iconify</RichLink>  
  可以一次查詢 Material Icons、Fontawesome、Mingcute 等各種 icon 集合的大大集合（？）
- <RichLink href="https://react-scan.com/" imgSrc="https://react-scan.million.dev/banner.png">React Scan</RichLink>  
  可以在每次 re-render 時高亮元件的變化，對於 debug 與優化效能非常有幫助
- [Easing Graphs](https://www.easing.dev/)  
  可以視覺化調整 easing 函數曲線
